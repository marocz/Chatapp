Extract first Character

Data sources in Power BI Desktop

Extracting data or values from a column is quite easy and can be done using the DAX or a nice GUI.

So we would be looking at how to extract data or character using the GUI Interface




The steps below is an example of using a delimiter in the extract columns function.





  1. Open the Query editor and select Add Columns from the menu above.



  2. On the Ribbon, select the Extract Dropdown. At this point the specifc action to be carried out is selected from the below:





  3. Select the actions Text After Delimiter



  4. Enter the required delimiter in the field titled Delimiter



  5. Enter the range of which the action is to be performed or applied.
